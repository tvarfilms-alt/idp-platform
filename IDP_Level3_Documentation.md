# IDP Level 3 — Cross-Section / Alpha Layer

## Назначение

Level 3 — это количественный скрининг акций IMOEX с автоматической генерацией сигналов BUY / HOLD / AVOID на основе трёх факторов: моментума, относительной стоимости и качества.

Входит в трёхуровневую иерархию:
- **L1 Macro Regime** → макро-контекст (Expansion / Slowdown / Recession)
- **L2 Market Risk** → режим рынка (Risk-on / Neutral / Risk-off)
- **L3 Cross-Section** → отбор конкретных акций + облигационная аллокация

## Скоринговая модель

### IdeaScore

```
IdeaScore = Momentum × 0.40 + RV × 0.35 + Quality × 0.25
```
